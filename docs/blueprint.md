# **App Name**: B2B Marketplace

## Core Features:

- Public Listings Browser: Enable unregistered users to browse listings with search and filter capabilities.
- Member Area Gating: Implement authentication to gate access to enriched listing content for logged-in members.
- Authentication and Sessions: Allow users to register, log in, and maintain sessions with email/password authentication.
- User Profile Management: Enable users to create and update their profiles, specifying account type (individual or company) and other relevant details.
- Admin Listing Management: Provide an admin interface for creating, reading, updating, and deleting listings with media upload capabilities.
- AI-Powered Listing Enrichment Tool: Automatically enhance listing descriptions using a tool, incorporating relevant information based on product category, specs, and existing user data to maximize engagement and conversion rates.
- Search and Filter Logic: Implement advanced filtering options including category, brand, year, location and more.

## Style Guidelines:

- Primary color: Deep blue (#385EA8) to convey trust and professionalism, aligning with a B2B aesthetic.
- Background color: Light gray (#F0F2F5), providing a clean, neutral backdrop that ensures readability and focus on content.
- Accent color: Muted orange (#D98E52) to highlight CTAs and important interactive elements without overwhelming the user.
- Headline font: 'Space Grotesk' sans-serif for headlines to convey a contemporary and high-tech look.
- Body font: 'Inter' sans-serif for body text to maintain readability and a clean user experience.
- Crisp, minimalist icons for categories and actions, focusing on clarity and ease of use.
- Modern spacing and soft shadows to create depth and visual appeal, enhancing the premium feel.
- Subtle animations and hover states to provide feedback and improve user interaction without being distracting.
